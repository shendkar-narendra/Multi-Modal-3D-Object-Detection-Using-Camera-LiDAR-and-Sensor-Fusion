# Multi-Modal-3D-Object-Detection-Using-Camera-LiDAR-and-Sensor-Fusion

Implemented and benchmarked state of the art object detection frameworks based on published research literature, including YOLOv8m for image based detection and PointPillars for LiDAR based 3D object detection. Evaluated the performance of camera only, LiDAR only, and camera–LiDAR fusion approaches on autonomous driving datasets using standard detection metrics.
